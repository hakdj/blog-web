# 📘 초보자용 데이터베이스 설정 가이드

## 🎯 목표
블로그 글 자동화 서비스를 위한 데이터베이스 만들기

## 📋 필요한 파일 (2개만!)

### 1️⃣ `SIMPLE_START.sql` - 실행 파일
이것 하나만 실행하면 모든 게 만들어집니다!

### 2️⃣ `CHECK_RESULT.sql` - 확인 파일  
실행 후 결과 확인용

---

## 🚀 실행 방법 (초간단!)

### 1단계: Supabase 열기
1. https://app.supabase.com 접속
2. 프로젝트 선택
3. 왼쪽 메뉴에서 **"SQL Editor"** 클릭
4. **"New Query"** 버튼 클릭 (또는 + 버튼)

### 2단계: SQL 실행
1. `SIMPLE_START.sql` 파일 열기
2. **전체 내용 복사** (Ctrl+A → Ctrl+C)
3. Supabase SQL Editor에 **붙여넣기** (Ctrl+V)
4. 오른쪽 하단 **"Run"** 버튼 클릭 (또는 Ctrl+Enter)

### 3단계: 확인
1. `CHECK_RESULT.sql` 파일 열기
2. 내용 복사해서 SQL Editor에 붙여넣기
3. **"Run"** 버튼 클릭
4. 결과 확인:
   - ✅ 플랜 6개 보임 = 성공!
   - ✅ 테이블 2개 보임 = 성공!
   - ✅ RLS 활성화됨 = 성공!

## ⚠️ 에러가 나면?

### "relation already exists" 에러
→ 이미 테이블이 있다는 뜻
→ `SIMPLE_START.sql`이 테이블을 삭제하고 새로 만들기 때문에 괜찮음
→ 그냥 무시하고 다시 실행

### 다른 에러
→ 에러 메시지 전체를 복사해서 알려주세요!
