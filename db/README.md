# 📁 데이터베이스 파일 가이드

## ✅ 필요한 파일 (2개만!)

### 1️⃣ `SIMPLE_START.sql` - 실행 파일 ⭐
**이것 하나만 실행하면 모든 게 만들어집니다!**

**하는 일:**
- plans 테이블 만들기 (Basic/Premium/Ultra × 월간/연간)
- blog_posts 테이블 만들기
- usage_logs 테이블 만들기
- 보안 설정 (RLS)
- 모든 정책 설정

### 2️⃣ `CHECK_RESULT.sql` - 확인 파일
실행 후 결과 확인용

---

## 🚀 사용 방법

1. Supabase → SQL Editor 열기
2. `SIMPLE_START.sql` 내용 복사
3. SQL Editor에 붙여넣기
4. **"Run"** 버튼 클릭
5. `CHECK_RESULT.sql` 실행해서 확인

---

## 📝 참고

- **에러가 나면**: 이미 만들어진 것들이 있어서 나는 에러일 수 있습니다. 무시하고 다시 실행하면 됩니다.
- **경고가 나면**: "Run this query" 버튼 클릭하고 진행하세요.
