# Supabase 저장된 쿼리 정리 가이드

## 🤔 PRIVATE에 있는 6개 쿼리 파일들

이것들은 **저장된 텍스트 파일**입니다:
- ✅ 데이터베이스에 영향 없음
- ✅ 삭제해도 문제 없음
- ✅ 참고용으로만 사용

## 🗑️ 삭제하는 방법 (선택사항)

### 방법 1: 하나씩 삭제
1. 왼쪽 PRIVATE 섹션에서 각 쿼리 옆 **"..." (점 3개)** 클릭
2. **"Delete"** 선택
3. 확인

### 방법 2: 그냥 무시
- 신경 안 쓰고 `SIMPLE_START.sql`만 사용
- 저장된 쿼리는 자동으로 실행되지 않음

## ✅ 추천 방법

**그냥 무시하세요!**
- 저장된 쿼리는 자동으로 실행 안 됨
- 새 쿼리 탭 열어서 `SIMPLE_START.sql`만 실행하면 됨
- 혼란스러우면 삭제해도 됨 (데이터베이스 영향 없음)

## 🎯 핵심

**중요한 건 `SIMPLE_START.sql` 실행하는 것!**
저장된 쿼리들은 그냥 참고용 파일이므로 무시해도 됩니다.
